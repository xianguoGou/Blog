网络

1. http 请求头 由三部分组成: 请求行、请求头、请求体

  • 请求行: GET /index.html HTTP/1.1
  • 请求头: Host: www.example.com
  • 请求体: {a: 1, b: 2}

2. http 响应头 由三部分组成: 响应行、响应头、响应体

  • 响应行: HTTP/1.1 200 OK
  • 响应头: Content-Type: text/html
  • 响应体: <html><body>Hello, World!</body></html>

2.1 常见的相应头：

  • Content-Type: 响应体的类型, 主要包括以下内容：
    • text/plain: 普通的纯文本，浏览器通常会以纯文本方式显示
    • text/html: HTML 文档，浏览器通常会将其渲染为网页
    • image/jpeg: 浏览器会将它视为 JPG 图片
    • application/json: JSON 数据
    • text/css: CSS 样式表, 浏览器通常会将其渲染为样式
    • application/javascript 或 text/javascript : js 代码, 浏览器通常会使用 js 执行引擎将它解析执行
    • attachment: 附件，浏览器看到这个类型，通常会触发下载功能
    • MIME 类型: 自定义的 MIME 类型
